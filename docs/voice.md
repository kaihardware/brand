# Voice and Language

KAI sounds observant, precise, warm, and quietly confident. Use concrete benefits, sensory language, evidence-backed claims, and plain tradeoffs. Avoid “revolutionary,” unqualified superlatives, AI-as-magic, manufactured urgency, or presenting prototypes as finished products.

Japanese: **好きな道具を、もっと好きにする。**  
English: **Refining the tools you love.**

The English line is canonical brand language, not a literal translation. Do not casually rewrite either line.
