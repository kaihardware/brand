# KAI Brand

> 好きな道具を、もっと好きにする。  
> Refining the tools you love.

This repository is KAI's public source of truth for what we believe, how we make product decisions, how we speak, and how approved assets are used.

KAI is a hardware company that uses AI to improve everyday tools. We do not add technology for spectacle. We refine the relationship between a person and a tool they already chose to care about.

## Change policy

Principles change rarely and require a brand owner and product owner. Substantive changes explain why in the pull request. Unless stated otherwise, KAI names, marks, and assets are not licensed for third-party use; licensing remains an explicit bootstrap decision.
